* Workload profiles

1. Docked external display + USB-C/TB peripherals (i915/xe, Type-C, UCSI)
2. Battery / TLP balanced vs performance (thermald, intel_pstate)
3. Wi-Fi roaming / VPN (iwlwifi, NetworkManager, openfortivpn)
4. Video conference (SOF audio, PipeWire, optional v4l2loopback)

* Acceptance (software bar without physical soak)
